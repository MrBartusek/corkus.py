.. _endpoints:

Corkus Endpoints
================

.. py:currentmodule:: corkus

Endpoints are classes designed for accessing API resources.
There are more or less similar to resources in `Official Wynncraft Documentation <https://docs.wynncraft.com>`_.

.. note::
    These classes are not meant to be initialized manually.
    Please use properties of :py:class:`Corkus`.
